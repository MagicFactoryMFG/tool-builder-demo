Don't have a Claude API key? Here's how to get one →
  1. Go to console.anthropic.com and sign in (or create a free account).
  2. Add a little credit under Settings → Billing (a few dollars is plenty — each scan costs a few cents).
  3. Open Settings → API keys and click Create Key.
  4. Copy the key (it starts with sk-ant-) and paste it in the field above.

Your key is saved only in this browser and is sent straight to Anthropic — Magic Factory never sees it. Usage is billed to your own Anthropic account.
